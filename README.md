# BEH App

This is a BEH app created with [bun create-beh](https://github.com/DanielCaz/create-beh).

## Getting Started

Run the server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- [Bun](https://bun.sh/)
- [Elysia][https://elysiajs.com/]
- [HTMX](https://htmx.org/)
